# 건강덩균 TEST

건이, 강이, 덩이, 균이 문장을 고르면 가장 많이 선택한 캐릭터를 결과로 보여주는 정적 웹앱입니다.

## 노트북 없이 사용

학생들이 노트북 없이 휴대폰으로 바로 쓰려면 이 폴더를 정적 웹사이트로 배포한 뒤 URL을 QR 코드로 공유하면 됩니다.

추천 방식:

1. Netlify Drop, GitHub Pages, Cloudflare Pages 같은 정적 호스팅에 이 폴더를 올립니다.
2. 배포된 주소의 `index.html`을 엽니다.
3. 그 주소로 QR 코드를 만들어 학생들에게 공유합니다.

학생 휴대폰에서 한 번 접속한 뒤에는 브라우저가 필요한 파일을 캐시합니다. 같은 주소로 다시 열면 인터넷 연결이 약해져도 이미 받은 화면은 계속 사용할 수 있습니다.

인터넷도 없고 노트북도 없다면 새 학생 휴대폰에 파일을 전달할 경로가 없어서 현장에서 바로 배포할 수 없습니다. 그 경우에는 수업 전에 URL로 접속시켜 두거나, 노트북/태블릿/휴대폰 중 하나를 파일 제공 장치로 써야 합니다.

## 단일 파일

`offline.html`은 이미지, CSS, JavaScript가 모두 들어 있는 단일 파일입니다. 호스팅 서비스에 파일 하나만 올려야 할 때 이 파일을 쓰면 됩니다.

소스 파일을 수정한 뒤 단일 파일을 다시 만들려면 아래 명령을 실행합니다.

```sh
node tools/build-offline.mjs
```
