# 건강덩균 TEST 사용법

## 노트북 없이 사용하기

노트북 없이 학생 휴대폰에서 바로 쓰려면 인터넷에 올린 주소가 필요합니다.

1. 이 폴더를 Netlify Drop, GitHub Pages, Cloudflare Pages 같은 정적 호스팅에 올립니다.
2. 배포된 주소의 `index.html`을 엽니다.
3. 그 주소로 QR 코드를 만들어 학생들에게 공유합니다.

학생 휴대폰에서 한 번 접속하면 브라우저가 필요한 파일을 캐시합니다. 같은 주소로 다시 열면 인터넷 연결이 약해져도 이미 받은 화면은 계속 사용할 수 있습니다.

`offline.html`은 이미지, CSS, JavaScript가 모두 들어 있는 단일 파일입니다. 폴더 전체를 올리기 어렵다면 `offline.html` 하나만 올려서 공유할 수 있습니다.

인터넷도 없고 노트북도 없다면 새 학생 휴대폰에 파일을 전달할 경로가 없어서 현장에서 바로 배포할 수 없습니다. 이 경우에는 수업 전에 URL로 접속시켜 두거나, 노트북/태블릿/휴대폰 중 하나를 파일 제공 장치로 써야 합니다.

## 인터넷 없는 강의실에서 노트북으로 사용하기

강의실 인터넷이 없어도 노트북과 학생 휴대폰이 같은 와이파이 또는 핫스팟에 연결되어 있으면 사용할 수 있습니다.

1. 노트북에서 이 폴더를 엽니다.
2. 터미널에서 아래 명령을 실행합니다.

```sh
python3 -m http.server 8062
```

3. 노트북의 와이파이 IP 주소를 확인합니다.

```sh
ipconfig getifaddr en0
```

4. 학생들에게 공유할 QR 주소는 아래 형식입니다.

```text
http://노트북IP주소:8062/offline.html
```

예를 들어 IP 주소가 `192.168.0.23`이면 QR 주소는 다음과 같습니다.

```text
http://192.168.0.23:8062/offline.html
```

`offline.html`은 이미지, CSS, JavaScript가 모두 한 파일에 포함된 강의용 파일입니다.
